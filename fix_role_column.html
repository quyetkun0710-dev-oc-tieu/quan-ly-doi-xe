🔧 Sửa Lỗi Đăng Nhập - Thêm Cột Role

⚠️ Vấn đề: Bảng taixe trong Supabase thiếu cột role, gây lỗi đăng nhập.

Bước 1: Mở Supabase SQL Editor

Vào supabase.com/dashboard → Chọn Project → SQL Editor → New Query

Bước 2: Dán và chạy SQL bên dưới

✅ Đã copy! 
-- =============================================
-- BƯỚC 1: Thêm cột role vào bảng taixe
-- =============================================
ALTER TABLE public.taixe ADD COLUMN IF NOT EXISTS role TEXT DEFAULT 'taixe';

-- =============================================
-- BƯỚC 2: Cập nhật role cho tài khoản quản lý
-- =============================================
UPDATE public.taixe SET role = 'manager' WHERE ma_nv IN ('JT071095', 'JT221196', 'JT01236');
UPDATE public.taixe SET role = 'admin' WHERE ma_nv IN ('483');

-- =============================================
-- BƯỚC 3: Thêm cột mat_khau (mật khẩu) nếu chưa có
-- =============================================
ALTER TABLE public.taixe ADD COLUMN IF NOT EXISTS mat_khau TEXT;

-- Đặt mật khẩu mặc định = admin123 cho tất cả
UPDATE public.taixe SET mat_khau = 'admin123' WHERE mat_khau IS NULL;

-- =============================================
-- KIỂM TRA KẾT QUẢ
-- =============================================
SELECT ma_nv, ho_ten, role, mat_khau FROM public.taixe ORDER BY role;

Bước 3: Đăng nhập thử

Tài khoản Mật khẩu Quyền
JT071095 JT071095 (hoặc admin123) Manager
JT221196 JT221196 (hoặc admin123) Manager
483 483 (hoặc admin123) Admin
506 506 (hoặc admin123) Tài xế

* Mật khẩu là mã NV của chính người đó, hoặc admin123

ℹ️ Lưu ý quan trọng

File index.html đã được cập nhật để hoạt động ngay cả khi chưa có cột role. Hệ thống sẽ tự xác định quyền dựa trên mã NV. Sau khi chạy SQL trên, việc đăng nhập sẽ ổn định hơn.
